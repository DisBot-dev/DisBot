Prefix
--------------
The prefix is `%`, type `%help` for the list of commands!

Pycord lib
---------------
`Github <https://github.com/Pycord-Development/pycord>`_
Install:
.. code:: sh

    # Linux/macOS
    python3 -m pip install -U py-cord

    # Windows
    py -3 -m pip install -U py-cord
